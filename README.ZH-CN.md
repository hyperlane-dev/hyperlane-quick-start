## 运行

### 运行

```sh
cargo run
```

### 在后台运行

```sh
cargo run -- -d
```

### 停止

```sh
cargo run stop
```

### 重启

```sh
cargo run restart
```

### 重启在后台运行

```sh
cargo run restart -d
```
